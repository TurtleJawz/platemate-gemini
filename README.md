# PlateMate (A Food Ingredient Detection and Recipe Generation App)
PlateMate is an Android application built to capture an image of leftover ingredients to output a list of food recipes (Meals, snacks, and/or drinks) that have the captured ingredients.

This is a thesis capstone project for the completion of our Computer Science course. 

# Resources/Tools used:

## For Gemini Version

*Gemini AI as computer vision model

*Spoonacular for recipe database

*Flutter and Dart for Android application development
